# 🦢 Foldly

An interactive origami game built with **Phaser** and **Devvit**, allowing Reddit users to learn and enjoy paper folding through step-by-step gameplay.

## ✨ Features

- 📄 Interactive origami folding experience
- 🎮 Built with Phaser 3
- 🔥 Daily streak system
- ⭐ Points and progression
- 🗺️ Journey tracking
- 📱 Responsive UI
- 🎨 Multiple origami models
- 🚀 Runs directly inside Reddit using Devvit

## 🛠️ Tech Stack

- Devvit
- Phaser 3
- TypeScript
- Vite
- Hono


## 🚀 Getting Started

### Prerequisites

- Node.js 22+
- Devvit CLI
- Reddit Developer Account

### Installation

```bash
git clone https://github.com/yourusername/foldly.git
cd foldly
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Deploy

```bash
npm run deploy
```

### Publish

```bash
npm run launch
```

## 🎮 Gameplay

1. Select an origami model.
2. Follow the folding instructions.
3. Complete the model.
4. Earn points.
5. Maintain your daily streak.
6. Unlock additional models through continued play.
